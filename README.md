# MGCE Elite Marketplace (Static Edition)

A premium, modern directory-style marketplace for **Maseno Girl Child Empowerment (MGCE)**. This platform focuses on ultra-fast performance, zero-dependency hosting, and hardened session security.

## Tech Stack

- **Core**: HTML5 & Vanilla JavaScript ES6+
- **Styling**: Tailwind CSS & Modern Vanilla CSS
- **Interactivity**: Alpine.js (Lightweight reactive UI)
- **Hardened Storage**: Custom IndexedDB Engine with Phone Trimming and Session Resilience
- **Icons**: Lucide Icons
- **Deployment**: Zero-build Static hosting (GitHub Pages, Netlify, Vercel)

## Key Features

- **Hardened Session Logic**: 30-second grace periods, silent profile healing, and 3-strike security pulses to prevent "Ghost Logouts."
- **Zero-Fail Favorites**: "Trust the Click" logic with LocalStorage backups. Hearts turn red instantly.
- **Popularity Metrics**: Real-time heart counts displayed globally on cards and detail views.
- **Elite Performance**: Zero-build architecture. Instant load times with no framework overhead.

## Getting Started

1. Simply clone or download this repository.
2. Open **`index.html`** in any modern web browser.
3. Start browsing and trading!

## Deployment

This project is purely static. Simply upload the root folder and the `assets/` directory to any static file host.
